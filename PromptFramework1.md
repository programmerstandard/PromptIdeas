# A-P-E

#### ( _Action_Purpose_Expectation_ 

#### or 

#### _Activity_Purpose_Expectation_ )

## Format

- State the [Action or Activity].
- Create a [Purpose].
- Define the [Expectation].

# C-A-R-E

#### ( _Context_Action_Rules_ 

#### and 

#### _Result_Example_ )


## Ordinary Format

- Describe the [Context]
- Write your [Action]
- Provide the [Rules], [Requirements] and Clarify for [Result]
- Give the [Example]

### Example 

```lisp
Analyze this sales dataset and show trends over the past year 
```

### Example Format 

```lisp
[Action] [Example] and [Results] [Context]
```

* Action : Analyze.
* Example: this sales dataset.
* Results : show trends.
* Context : over the past year.

# R-T-F

## Format

- Act as [Role]
- Describe the [Task]
- Show as [format]
