# A-P-E

### ( _Action_Purpose_Expectation_ or _Activity_Purpose_Expectation_ )

## Format

- State the [Action or Activity].
- Create a [Purpose].
- Define the [Expectation].

# C-A-R-E

## Format

- Describe the [Context]
- Write your [Action]
- Provide the [Rules] and Clarify for [Result]
- Give the [Example]

# R-T-F

## Format

- Act as [Role]
- Describe the [Task]
- Show as [format]
